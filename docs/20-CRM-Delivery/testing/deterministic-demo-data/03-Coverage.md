# 覆盖率：固定种子演示数据

| 范围 | 覆盖 |
|---|---|
| 操作模式 | check、generate，2/2 |
| 广覆盖聚合 | 20 类核心/关联数据全部命中，每类 50 条 |
| 产品链 | 产品、商机产品、合同产品 3/3，各 50 条 |
| 引用校验 | 联系人、合同、财务 3/3，孤儿 0 |
| 状态不变量 | 2/2 |
| cleanup | 1/1，残留 0 |
| 部署策略 | preserve、insert、replace 三模式门禁；本机 replace 后恢复 preserve |

本项是 Bash/SQL 集成资产，不进入 Java JaCoCo。审批业务事实只作为导入历史，不伪造 Flowable 引擎实例。
