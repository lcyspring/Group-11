# 统计指标血缘与刷新测试计划

## 后端自动化

- 验证显式配置可绑定刷新、权限、重算、对账及嵌套指标字段；
- 验证按 scope 查询只返回当前域指标，不混入其他统计域；
- 验证同一域重复指标编码在应用启动阶段失败；
- 验证六个 scope 均绑定各自原有统计权限；
- 验证接口 scope 白名单不接受未接入的任意值。

## 前端与构建

- 六个统计入口接入同一共享组件；
- 手动刷新只调用当前激活图表并更新时间；
- 元数据失败只显示告警，不阻断原统计查询；
- 专项 ESLint 以零警告通过；
- Server 和 Web production build 均在 Ubuntu 26.04 容器完成。

## 运行验收

- 使用有权账号分别请求六个目录，业务码应为 0 且 scope 唯一；
- 无令牌请求应返回统一业务码 401，非法或无权 scope 应被拒绝；
- 页面展开目录时可看到来源表、字段、业务时间、公式、过滤和权限。
