# 测试计划

## 后端

1. Service 将选择的用户/部门范围转换为负责人 ID 后传入 Mapper；
2. 空负责人范围时返回空集合，不生成非法 `IN ()`；
3. SQL 排除已删除客户、已删除合同和非审批通过合同；
4. SQL 使用合同下单时间筛选；
5. SQL 按成交金额降序、客户 ID 升序并限制 10 条；
6. CRM 全量测试并生成 JaCoCo。

## 前端

1. 客户统计页可加载 TOP10 页签；
2. 柱状图保持服务端排名第一的客户位于顶部；
3. 表格显示排名、客户、合同数和成交金额；
4. 客户名称可钻取客户详情；
5. 中、英、阿三语 key 完整；
6. Ubuntu 26.04 Vite production build。
