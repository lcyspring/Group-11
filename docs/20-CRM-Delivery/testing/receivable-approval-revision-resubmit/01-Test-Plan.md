# 测试计划

日期：2026-07-14

## 范围

1. 回款草稿、审批中、通过、驳回、取消的编辑、删除和重提边界；
2. 回款创建/提交及回款计划创建的对象权限契约；
3. 合同锁、回款锁、提交额度复核和新流程实例写入；
4. 重复、旧流程、乱序和并发终态回调；
5. 回款/计划金额 `0`、负数和 `0.01` 边界；
6. 未审批通过合同创建回款计划的后端拒绝与前端禁用；
7. 已回款汇总只统计审批通过状态；
8. 前端修订、查看流程和删除入口及 production build。

## 环境

统一使用 `localhost/mitedtsm-build-ubuntu:26.04`，不调用宿主机 JDK、Node 或 pnpm。
