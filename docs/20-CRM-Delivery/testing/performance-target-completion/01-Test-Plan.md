# 测试计划

## 服务与 SQL

- 五类指标分发到正确 Mapper；
- 合同/回款只统计审批通过单据，并使用各自业务时间；
- 跟进只统计客户类记录并按创建人归属；
- 新增客户/商机按负责人和创建时间聚合；
- 全部实际值 SQL 排除逻辑删除；
- 目标、实际、月度完成率和年度完成率可复算；
- 目标为 0 时月度/年度完成率为空；
- 部门/个人筛选和目标范围不一致时拒绝；
- 非完整自然年查询拒绝。

## 前端

- 五类指标可切换；
- 目标、实际、完成率图表和表格共用同一响应；
- 年度摘要由后端精确汇总，不在浏览器用浮点数重新累计；
- 大额数值以字符串分组格式化；
- 中、英、阿文案、Prettier、ESLint、production build；
- CRM 统计纯函数测试可在 Ubuntu 发行版 Node 中执行。
