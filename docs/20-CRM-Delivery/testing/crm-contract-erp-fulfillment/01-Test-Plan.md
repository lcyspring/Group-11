# 测试计划

日期：2026-07-16。分支：`develop`。

1. 未审批、未签署合同不得创建履约订单，并返回可展示的阻塞码。
2. 缺客户映射、缺产品映射分别阻塞；映射保存后合同转为可履约。
3. USD 合同按 YAML 冻结汇率换算为 CNY，金额保持六位精度。
4. 首次创建冻结 JSON 和 SHA-256；重复创建返回同一 ERP 订单。
5. ERP 已创建但 CRM 回写失败时，重试按双侧唯一键取回原订单。
6. 外部来源 ERP 订单不可编辑客户/产品/价格行，不可删除，但可审核。
7. ERP 审核、出库量、退货量可自动回传，并可显式刷新补偿。
8. SQL 迁移连续执行两次，表、字段、唯一键和菜单不重复。
9. Ubuntu 26.04 容器执行 CRM、ERP、Web 专项及生产构建。
10. 真实 API 验收结束后清理合同、映射、订单和 ERP 临时主数据，残留为 0。
