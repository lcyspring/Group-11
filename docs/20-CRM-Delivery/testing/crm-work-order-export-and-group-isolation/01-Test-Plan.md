# CRM 工单导出与处理组隔离测试计划

## 目标

验证工单筛选/勾选导出具备完整数据权限，并验证停用一个处理组不会全局禁用仍属于其他启用组的
客服人员。

## 用例

1. 未勾选时导出当前筛选条件下的全部授权工单；
2. 勾选时仅导出选中编号，且选中编号仍受数据权限约束；
3. 导出模型将类型、优先级、状态和来源编码转换成可读名称；
4. 导出权限与查询权限独立配置；
5. 原处理组和处理人都不变时禁止无意义改派；
6. 原处理组停用、目标组启用且处理人同时属于两组时，允许同一处理人跨组改派；
7. 目标组停用、不支持工单类型或人员不是目标组成员时仍拒绝改派；
8. 前端中、英、阿语言与代码规范检查通过。
9. 导出权限必须按组件唯一定位客服工单页面，不得误挂到同路径的工单统计页面。
