# 客户公海联系人归属测试计划

1. 客户入池时在删除客户 OWNER 权限之前清空联系人负责人；
2. 客户领取/分配时恢复全部联系人负责人；
3. 清空负责人只删除 OWNER 权限，保留 READ/WRITE；
4. 指定新负责人时，已存在的团队权限升级为 OWNER，删除旧 OWNER；
5. 联系人归属、权限、客户归属和公海记录任一失败时整体回滚；
6. Ubuntu 26.04 中执行 CRM 全量测试、JaCoCo 和 Server 生产构建；
7. 通过显式 KDL 替换 Server，保持 Web/Mall 可观察。
8. 兼容迁移运行两次，客户范围内归属与 OWNER 权限漂移均为 0。
9. 迁移前后客户 OWNER 权限指纹一致，孤立联系人保持不变。
