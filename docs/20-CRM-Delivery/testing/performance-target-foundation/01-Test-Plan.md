# 测试计划

## 自动化范围

- 请求必须包含合法范围、合法指标、年度和正好 12 个月目标；
- 每个月均验证非空、非负和小数位；
- 公司范围编号只能为 0；
- 部门和人员范围调用 System API 有效性校验；
- 计数指标拒绝小数；
- 已有月份保持或更新，缺失月份创建；
- 季度和年度只从月份派生；
- MySQL 8 迁移可执行且可重复执行；
- 逻辑删除后可以重新创建同一目标；
- 权限和中、英、阿菜单记录重复执行不增加重复数据。

## 回归范围

- Ubuntu 26.04 容器内 CRM 全量自动化；
- Maven 20 模块 reactor；
- JaCoCo 全模块覆盖率；
- Shell 语法、Git whitespace 和 UTF-8。
