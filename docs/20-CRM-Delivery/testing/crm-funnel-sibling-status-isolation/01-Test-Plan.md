# 销售漏斗同级状态隔离测试计划

## 目标

验证商机在同一状态组内变更状态时，只有目标状态的当前存量发生变化，其它同级状态不被累计；
同时保证点击状态查看的明细与该状态统计数量采用同一精确口径。

## 用例

1. 给定三个同级状态，其当前商机数分别为 `10 / 4 / 1`，统计结果必须保持 `10 / 4 / 1`；
2. 禁止把三个状态反向累计成 `15 / 5 / 1`；
3. 金额按当前状态独立汇总，不跨状态累计；
4. 状态详情查询同时限定状态组、状态 ID、人员范围和时间范围；
5. 状态详情不得再使用状态排序的 `>=` 范围条件；
6. CRM 全量回归测试不得出现失败或跳过。
