# 测试计划

日期：2026-07-14

## 范围

1. BPM 通过、驳回、取消到 CRM 状态的映射；
2. 草稿、审批中、通过、驳回、取消的编辑和删除边界；
3. 驳回/取消合同修订回草稿及新流程重提；
4. 提交接口的 WRITE 对象权限契约和重复提交行锁；
5. 重复回调、旧流程回调、乱序回调和并发终态条件更新；
6. 商机转换合同修订时来源、客户和负责人不可变；
7. 前端修订入口、删除入口、三语文案及 production build。

## 环境

统一使用 `localhost/mitedtsm-build-ubuntu:26.04`，不调用宿主机 JDK、Node 或 pnpm。
