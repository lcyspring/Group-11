# CRM 客户关怀测试计划

- 验证生日、节假日、成交后定期回访三类规则及字段归一化；
- 验证真实日期、渠道模板和回访天数边界；
- 验证计划详情、编辑、启停和启用计划禁止删除；
- 验证同意回退、退订、空地址、频控、record-only 与 system 提供商结果；
- 验证记录和生日查询的 CRM 客户数据范围及可读名称；
- 验证客户与联系人新增/编辑均可维护生日；查询通过显式对象类型分别返回客户自身生日或联系人生日；
- 验证计划/事件/渠道幂等、调度分布式锁和中断处理；
- 在 Ubuntu 26.04 容器执行 CRM 全量、JaCoCo、Web 专项、ESLint 和生产构建；
- 使用显式 YAML 对真实 API 和 MySQL 数据清理进行验收。
