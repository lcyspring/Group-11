# BPM 自定义审批详情测试计划

1. 校验浏览器列表路由、历史 `/index` 路径和无扩展名路径被前端拒绝。
2. 校验只有完整 `.vue` 与 `.tsx` 组件路径能精确解析，无扩展名 Vue 路径必须失败。
3. 校验相对路径、协议相对路径和空路径被拒绝。
4. 校验组件注册使用模块键精确查找，不再使用子串匹配。
5. 校验受管 CRM BPM 示例区分浏览器发起路由和审批详情组件。
6. 校验回款、合同详情接受流程 `businessKey` 的字符串 ID。
7. 校验报销、退款提供可嵌入审批页的单条详情组件。
8. 在 Ubuntu 26.04 容器中执行专项测试、ESLint 和 Web 生产构建。
9. 校验存量流程定义路径由幂等修复 SQL 更新，并留下定义级审计记录。
