## 覆盖场景

- 日、周、月周期归一和周期唯一性；
- 接收人去重、排除作者、至少一名有效接收人；
- 草稿维护、提交锁定和越权读取；
- 我的报告/收到的报告双视图；
- 操作菜单、附件和三语言字段；
- `LocalDate` 响应必须是 `YYYY-MM-DD`，不能返回数组。
