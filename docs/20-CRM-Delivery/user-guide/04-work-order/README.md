# 客服工单验收

1. 在 `客服处理组`（`/crm/work-order-group`）配置处理组、成员和自动派单范围；
2. 在 `客服工单`（`/crm/work-order`）从客户或合同创建工单，填写类型、优先级、描述和附件；
3. 验证手工分派、最小负载自动派单、未分配领取、退回、抄送和跨组权限；
4. 移动签到校验经纬度、围栏和签到轨迹；越界签到应失败；
5. 验证 SLA 工作日历、暂停/恢复、到期和自动升级；
6. 完结后状态轨迹、通知和客户 360 工单页同步更新；
7. 在 `数据统计 → 工单统计`（`/crm/statistics/work-order`）核对总量、状态、类型、处理人和趋势。

预期：同一旧版本并发处理只有一次成功；无组权限用户不能分派；所有动作均记录操作者、时间、旧
状态和新状态，页面操作后无需手工刷新。
