# CRM-ACTIVITY-BUG-002：超时任务无法继续开始处理

- 模块：CRM / 活动 / 超时任务
- 状态：已关闭
- 分支：`develop`

## 现象与根因

调度器把到期未开始任务标记为超时后，开始命令只接受未开始状态，负责人再也不能实际处理该
任务。超时本应表达 SLA/时效事实，不应成为不可恢复的业务终态。

## 修复与验证

开始命令允许未开始或超时任务进入进行中，动作轨迹保留 `50 → 10`；完成、未完成、取消仍按
明确状态机校验。超时扫描继续使用租户分页和 Redisson 单执行锁。活动 Service、调度专项共
15 项及 CRM 354 项 Ubuntu 26.04 回归通过。
