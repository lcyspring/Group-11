# STAT-BUG-006：统计请求失败后 loading 不释放

发现日期：2026-07-13。关闭日期：2026-07-16。级别：P1。

部分统计组件在成功分支末尾关闭 loading，请求抛错时页面会永久保持骨架屏或表格遮罩。修复后
加载入口统一在 `try/finally` 中执行请求，并在 `finally` 无条件恢复 `loading=false`。

最终契约测试覆盖客户总量、客户转化率和商机汇总三个代表性组件，锁定“置 true → await 请求 →
finally 置 false”的控制流。Ubuntu 26.04 统计前端 10/10、ESLint 零警告、生产构建通过。
