# CRM-CONTRACT-BUG-008：本地登记适配器错误暴露电子签方式

发现日期：2026-07-14。状态：Fixed。

## 现象与风险

初版签署表单固定展示“线下签署”和“电子签”两个选项，服务端也只校验数值范围。当前
`local-record` 实际只记录线下事实，却会接受电子签方式，形成“已完成电子签”的虚假业务语义。

## 修复

`CrmContractSignProvider` 新增显式能力集合。服务端拒绝 Provider 不支持的方式，生命周期摘要返回
能力集合，前端据此生成单选项。`local-record` 仅返回方式 1；选择方式 2 返回业务码
`1020000022`。专项单测覆盖能力声明和拒绝分支。
