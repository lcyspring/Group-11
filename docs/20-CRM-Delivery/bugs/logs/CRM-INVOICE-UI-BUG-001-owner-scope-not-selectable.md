# CRM-INVOICE-UI-BUG-001：发票列表无法选择归属范围

状态：Fixed

日期：2026-07-15

分支：`develop`

## 现象

发票后端分页已经支持 `sceneType`，但前端从未提交该字段。普通用户因此固定落入“本人负责”默认范围，
无法查看直接参与、下属负责或角色组织范围内的发票，财务角色的部门数据范围无法在页面使用。

## 修复与验证

发票列表新增我负责、我参与、下属负责、组织范围四个 Tab，请求显式携带 `sceneType`；标签同步中、英、
阿三语。专项 ESLint 和 production build 纳入 Ubuntu 26.04 验证配置。
