# CRM-CORE-BUG-030：回款审批历史可被删除

- 日期：2026-07-14
- 级别：P0
- 关联：GAP-FIN-002、GAP-APR-002
- 状态：已关闭

## 现象与根因

旧逻辑只禁止删除审批通过或关联计划的回款；审批中、驳回、取消且无计划的回款仍可删除，
使 BPM businessKey 指向不存在记录并破坏审计历史，页面对所有状态也显示删除按钮。

## 修复关键

只允许删除从未提交、无流程实例且未关联计划的新草稿；列表和合同详情子列表使用相同条件
展示删除入口。

## 验证

专项覆盖驳回历史、计划关联草稿删除拒绝和无计划新草稿删除成功；CRM 143/143、Web build 通过。
