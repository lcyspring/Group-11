# STAT-BUG-030：待回款角标与默认列表口径不一致

- 日期：2026-07-14
- 级别：P1
- 关联：GAP-FIN-001、GAP-RPT-002
- 状态：已关闭

## 现象与根因

默认待回款列表按“提醒日已到”查询，角标却额外要求计划已经逾期，并使用严格小于今天，
导致用户进入待办后列表条数大于左侧角标。

## 修复关键

角标与默认列表复用同一查询条件：未产生审批通过回款，且提醒时间小于等于今天；逾期仍由
独立筛选按计划回款日判断。

## 验证

Mapper 契约断言提醒条件含 `remind_time` 且不混入 `return_time`；CRM 149/149。
