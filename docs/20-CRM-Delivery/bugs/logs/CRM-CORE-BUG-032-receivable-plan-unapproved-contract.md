# CRM-CORE-BUG-032：未生效合同可创建回款计划

- 日期：2026-07-14
- 级别：P0
- 关联：GAP-FIN-001、ADR-006
- 状态：已关闭

## 现象与根因

创建回款已经要求合同审批通过，但回款计划只读取合同并继承客户，不检查合同状态；前端
计划表单也列出全部合同。草稿、审批中、驳回或取消合同因此可提前产生财务计划。

## 修复关键

后端使用合同校验服务并要求 `APPROVE(20)`；计划创建同时要求合同 WRITE。前端下拉禁用
所有非审批通过合同。

## 验证

专项覆盖审批中合同拒绝和审批通过合同创建成功；CRM 143/143、Web build 通过。
