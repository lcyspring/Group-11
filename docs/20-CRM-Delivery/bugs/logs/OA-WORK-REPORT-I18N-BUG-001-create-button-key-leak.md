# OA-WORK-REPORT-I18N-BUG-001 工作报告新增按钮语言键泄漏

## 现象

工作报告列表的新增按钮显示 `common.create`，与页面其他中文文案不一致；新增弹窗标题则能正常显示。

## 根因

按钮误用了 CRM 语言域中不存在的通用 Key。弹窗已经使用 `workReport.create`，同一个操作存在两套语言引用。

## 修复

- 列表新增按钮统一使用 `workReport.create`；
- 复用已有中、英、阿三语言工作报告文案；
- 增加专项测试，禁止工作报告页面重新引用不存在的 `common.create`。

## 验收预期

中文显示“新建工作报告”，英文显示“Create Work Report”，阿拉伯语显示对应翻译；按钮与弹窗标题保持一致。
