# PODMAN-TEST-BUG-001：数据生成器测试误依赖工具链镜像未提供的 rg

## 现象

宿主机执行演示数据生成器契约测试通过，但在指定 Ubuntu 26.04 工具链镜像中执行时提示
`rg: command not found`，导致容器验收失败。

## 根因

测试只需要判断生成 SQL 是否残留模板占位符和是否包含关键断言，却使用了非基础命令 `rg`；工具链镜像
保证构建所需组件，不保证安装开发者宿主机上的所有搜索工具。

## 修复与验证

将测试内四处简单匹配改为 Ubuntu 基础环境已有的 `grep -E/-q`，不修改生成器和工具链镜像。
使用 `ghcr.io/elel-code/group-11-build-ubuntu:26.04` 重新执行语法检查、生成、checksum、占位符、
关键断言和非法基数拒绝测试，全部通过。
