# CRM-CONTRACT-FE-BUG-004：部分合同“更多”点击无反馈

日期：2026-07-18。分支：`develop`。状态：已修复。

## 现象

合同列表中部分记录的“更多”能够展示操作，部分记录点击后没有可见反馈。

## 原因

`TableActions` 只要检测到具名插槽就展示“更多”，但合同的编辑、提交审批、查看审批和删除均受
合同状态及权限控制。没有任何可用操作的记录仍显示了空下拉入口；模板中第一个下拉项缺少就地
闭合，后续结束标签错位，导致不同状态渲染结果不稳定。

## 修复

- 按审批状态、流程实例和当前用户权限计算是否存在扩展操作；
- 无可用操作时不展示“更多”，保留始终可用的详情入口；
- 修正下拉项的开始/结束标签配对。

## 回归

源码契约测试覆盖“更多”显隐、更新/删除权限和四类操作的状态门禁，并纳入 Ubuntu 26.04 Web
类型检查与生产构建。
