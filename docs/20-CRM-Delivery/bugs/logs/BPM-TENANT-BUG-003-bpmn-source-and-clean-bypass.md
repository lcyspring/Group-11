# BPM-TENANT-BUG-003：BPMN 原文与模型清理存在租户旁路

日期：2026-07-14

分支：`develop`

状态：已修复

## 现象与影响

模型 XML 原文读取和弃用更新接口直接操作 Flowable 字节源；模型清理又只按流程 Key 查询。
不同租户允许复用相同 Key，因此存在读取、覆盖或清理其他租户流程数据的风险。

## 修复与验证

- XML 读写先通过 `modelId + modelTenantId` 校验模型；
- BPMN 定义模型统一委托租户安全的流程定义服务；
- 运行实例、历史实例和任务清理均增加当前租户条件；
- 新增 XML 读写和 BPMN 委托测试，BPM 总回归 54 个、失败 0。
