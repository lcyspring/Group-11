# WEB-BUILD-BUG-006：语言分包验证器依赖旧打包文本形态

日期：2026-07-19。分支：`develop`。级别：P1/多语言交付。状态：已关闭。

## 现象与根因

旧验证器只在主入口文本中匹配固定的动态 import 代码形态。Vite 8 调整依赖图和 chunk 输出后，三语
分包仍可能正确生成，但验证器会误报；反过来只检查文件存在又无法证明入口可达。

## 修复关键

- 从产物目录识别每种语言唯一的 hash chunk，并保留最小体积校验；
- 检查三个分包互不复用同一文件；
- 扫描生产 chunk，要求至少一个由 `index.html` 可达的入口同时引用三种语言分包。

## 回归

验证结果：`zh-CN-CCtFuEKH.js`、`en-B5Ovujx8.js`、`ar-BejmiZG8.js` 均唯一、非空且从生产入口可达。
