# CRM-CORE-BUG-028：回款创建和提交缺少对象权限

- 日期：2026-07-14
- 级别：P0/安全
- 关联：GAP-IAM-003、GAP-FIN-001/002
- 状态：已关闭

## 现象与根因

回款和回款计划创建只有功能权限，未校验来源合同对象；回款提交也只有功能权限。拥有菜单
权限的用户可枚举合同或回款 ID，绕过对象团队范围创建财务子记录或发起审批。

## 修复关键

创建回款/计划要求来源合同 WRITE；提交回款要求目标回款 WRITE。权限校验放在应用服务，
不依赖页面隐藏。

## 验证

反射契约测试逐项断言业务类型、SpEL ID 和 WRITE 等级；回款权限专项通过，CRM 143/143。
真实双角色 HTTP 负向仍待完整鉴权数据集补证。
