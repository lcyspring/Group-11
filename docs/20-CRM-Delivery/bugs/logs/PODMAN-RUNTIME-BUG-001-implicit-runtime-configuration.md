# PODMAN-RUNTIME-BUG-001：运行配置来源分散且可被宿主环境隐式改变

## 发现

旧 `deploy.sh` / `stop.sh` / `image-archives.sh` 使用命令行选项、环境变量默认值及正文硬编码共同决定
模式、端口、镜像、卷、凭据、profile 和删除数据行为。

## 影响

- 无法仅凭一份配置复现部署；
- 宿主遗留环境变量可能静默改变运行结果；
- 删除卷使用快捷参数，操作意图缺少可审计配置；
- 凭据和健康检查参数散落，修改容易遗漏相关探测命令。

## 修复

引入必填 YAML 模板和共享读取库；三个运行辅助入口命令行只接收配置路径；删除旧运行
参数和环境变量覆盖；将运行行为、服务参数和健康检查统一读取自 YAML。

## 验证

结构化测试验证参数契约及 up/down `check`；Pod ID/状态前后一致。
