# BPM-EXPRESSION-I18N-BUG-005：流程表达式页面语言混用

## 现象

流程管理的流程表达式页面在切换语言后仍保留固定中文标题，阿拉伯语下字段键缺失并回退为其他语言或键名。

## 根因

- `doc-alert` 标题直接硬编码“流程表达式”；
- 阿拉伯语 BPM 资源未提供 `processExpression` 完整对象。

## 修复

- 标题统一使用 `bpm.processExpression.title`；
- 中文、英文、阿拉伯语均提供标题、字段、占位符和校验消息；
- 新增 BPM 前端专项 ESLint 门禁，避免模板重新出现固定语言。

## 验证

三份语言资源通过生产构建完整性检查，专项 ESLint 与 Ubuntu 26.04 Web 生产构建通过。
