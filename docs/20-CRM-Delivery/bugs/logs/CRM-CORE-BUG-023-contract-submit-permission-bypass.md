# CRM-CORE-BUG-023：合同提交审批缺少对象权限

- 日期：2026-07-14
- 级别：P0/安全
- 关联：GAP-IAM-003、GAP-APR-002
- 状态：已关闭

## 现象与根因

控制器只校验 `crm:contract:update` 功能权限，`submitContract` Service 没有合同对象权限
注解。拥有菜单功能但不属于目标合同团队的用户可尝试枚举合同 ID 发起审批。

## 修复关键

在提交应用服务增加 `CRM_CONTRACT + #id + WRITE` 对象权限；功能权限与对象权限同时满足
才可发起。提交逻辑同时锁定合同记录，防止同一合同重复请求创建多个流程实例。

## 验证

反射契约测试断言业务类型、SpEL ID 和 WRITE 等级；合同专项 15/15、CRM 全量 124/124。
真实未授权 HTTP 请求仍需随完整鉴权运行环境补集成证据。
