# MALL-H5-BUILD-BUG-002：Mall 自有 Sass 全局旧函数淹没构建日志

日期：2026-07-18。分支：`develop`。优先级：P2/维护。状态：已关闭。

## 现象与根因

H5 虽能生成产物，但构建日志达到 28732 行。`sheep/scss` 仍使用全局 `map-merge`、`map-get`、
`type-of`、`mix`、`desaturate`、`darken` 等函数；同一变量文件被组件重复注入后形成 895 条显式
旧函数告警和大量省略重复项，真正错误难以识别。

## 修复关键

- 按文件引入 `sass:map/list/meta/color`；
- 迁移 map、列表、类型和颜色函数，保持生成 CSS 语义；
- 增加源码负向门禁，禁止重新引入这些全局旧函数；
- 不热改 HBuilderX 镜像内部编译器和 `uni_modules`，剩余告警单独登记。
