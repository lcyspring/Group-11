# CRM-EXPORT-BUG-001：联系人和合同导出被分页截断

- 发现/关闭日期：2026-07-15
- 级别：P1 / 数据完整性

## 现象与根因

联系人导出把 `PAGE_SIZE_NONE` 误写入 `pageNo`，合同导出则完全没有取消分页。界面提示导出筛选结果，
实际文件只包含默认页或产生异常页码，且没有明显错误提醒。

## 修复关键

- 联系人改为 `setPageSize(PAGE_SIZE_NONE)`；
- 合同导出显式设置无分页；
- 两者在完整查询后继续执行对象级导出权限校验。

## 验证

Ubuntu 26.04 Server build、CRM 231/231 和 Web production build 通过；导出查询不再沿用默认分页大小。
