# 2026-07-15 公共线索运行验收

运行环境使用 rootless Podman，Server 和 Web 产物由 Ubuntu 26.04 构建容器生成。启动和热替换
命令行只指定显式 YAML 配置路径：

```bash
cd podman
bash ./up.sh ./config/runtime-local.yaml
bash ./up.sh ./config/runtime-local-rebuild-server.yaml
```
