## 真实 API 生命周期

无引用样本完成：创建 → 放入公海 → 转入垃圾池 → 垃圾池分页可见 → 恢复到公海 →
再次转入垃圾池 → 永久删除。恢复后 `poolStatus=PUBLIC`、负责人为空、原因
`RESTORE_PUBLIC`；永久删除后详情数据为空。

带首联系人样本首次永久删除返回 `1020006032`，消息明确指出“联系人”。管理员删除垃圾
池内唯一首联系人后再次永久删除成功，从而同时验证引用保护和解除阻塞规则。

验收期间临时把测试账号加入既有 `crm_admin` 角色，结束后通过系统权限接口恢复原角色集。
数据库核对临时授权为 0，有效测试客户和联系人均为 0。文档不记录访问令牌和账号口令。
