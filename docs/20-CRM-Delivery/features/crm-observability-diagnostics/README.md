# CRM 可观测诊断包

本功能把 CRM 单节点运行验收从“能打开页面”扩展为可重复诊断：

- 检查 Server health、Web/Mall HTTP 和七个业务容器运行状态；
- 记录容器重启次数、资源快照和最近窗口日志；
- 只读采集 MySQL 连接、线程、异常连接、表数和库大小；
- 记录宿主 CPU、内存和项目所在磁盘容量；
- 按 KDL 中 SLI 阈值输出 PASS/FAIL，并保留 Markdown 摘要和压缩诊断包；
- 原始日志与指标包只进入 Git ignored 本机目录，文档只提交脱敏结论。

脚本只接收一个 KDL 路径，不停止、重启或修改服务，也不写 CRM 业务数据。
