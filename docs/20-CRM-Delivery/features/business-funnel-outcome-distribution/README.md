# CRM-FEATURE-065：漏斗终态结果分布增强

日期：2026-07-18。分支：`develop`。状态：已实现。

## 背景

销售漏斗的阶段展示依赖所选商机状态组。状态组“测试一下”没有配置初始阶段时，阶段图缺失属于
配置数据问题，并非原漏斗统计错误。为提高可解释性，本增强在阶段漏斗之外增加终态结果分布。

## 能力

- 阶段漏斗继续表达“至少到达本阶段”的数量、金额和相邻转化率；
- 独立结果图稳定展示赢单、输单、无效，包括 0 值；
- 终态占比以全部结单商机为分母，三类结果均可钻取明细；
- 所选状态组没有任何有效阶段时，页面明确提示先配置初始阶段，不再只显示一个难以解释的图形。

## 测试

Mapper 契约覆盖三个终态，Service 覆盖阶段累计和结果占比，Web 契约覆盖双图、明细 API 与配置提示。
