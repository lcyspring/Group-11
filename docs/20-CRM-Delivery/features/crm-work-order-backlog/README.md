# CRM 客服工单待办接入

## 审计结论

项目原有 `crm/backlog` 只聚合今日联系、跟进、公海、合同/回款审批和提醒，
不包含客服工单，也不能据此声明“统一待办已完成”。

## 本批增强

- 新增“客服工单待办”入口和角标；
- 明确待办口径：
  - 当前用户作为处理人：待处理、处理中；
  - 当前用户作为创建人：已退回；
- 已完结工单自动退出待办；
- 查询始终使用当前登录用户，不因 `query-all` 权限扩大个人待办。

## 剩余项

Flowable 跨业务统一待办及其租户流程发布运行验收仍属于 GAP-APR-001/003，
本目录不将其误报为已完成。
