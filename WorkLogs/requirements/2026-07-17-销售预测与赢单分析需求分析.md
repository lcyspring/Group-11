# 销售预测与商机赢单分析需求分析

## 1. 需求概述

### 1.1 需求背景

CRM系统需要提供销售预测和商机赢单分析功能，帮助管理层了解销售趋势、预测未来业绩、分析赢单因素，从而做出更精准的决策。

### 1.2 需求来源

根据业务部门反馈，需要新增以下两个核心分析功能：
- **销售预测**：基于历史数据预测未来一段时间的销售额
- **商机赢单分析**：分析商机赢单的关键因素和趋势

## 2. 功能需求

### 2.1 销售预测接口

**功能描述**：根据历史商机数据，预测未来一段时间内的销售额。

**输入参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| deptId | Long | 是 | 部门ID |
| userId | Long | 否 | 用户ID，为空则查询部门下所有用户 |
| interval | Integer | 是 | 时间间隔类型（1-天，2-周，3-月） |
| times | LocalDateTime[] | 是 | 时间范围 |
| forecastPeriod | Integer | 是 | 预测周期（月） |

**输出参数**：
| 参数名 | 类型 | 说明 |
|--------|------|------|
| forecastData | List | 预测数据列表 |
| forecastData[].time | String | 预测时间点 |
| forecastData[].forecastAmount | BigDecimal | 预测金额 |
| forecastData[].actualAmount | BigDecimal | 实际金额（历史数据） |
| totalForecastAmount | BigDecimal | 总预测金额 |
| totalActualAmount | BigDecimal | 总实际金额 |

**业务逻辑**：
- 根据历史商机成交数据（赢单）计算平均成交金额
- 根据历史转化率计算预测成交数量
- 综合计算预测销售额

### 2.2 商机赢单分析接口

**功能描述**：分析商机赢单的关键因素，包括赢单率、赢单金额分布、行业分布、客户等级分布等。

**输入参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| deptId | Long | 是 | 部门ID |
| userId | Long | 否 | 用户ID，为空则查询部门下所有用户 |
| interval | Integer | 是 | 时间间隔类型 |
| times | LocalDateTime[] | 是 | 时间范围 |

**输出参数**：
| 参数名 | 类型 | 说明 |
|--------|------|------|
| winRate | BigDecimal | 赢单率（赢单数/总商机数） |
| totalBusinessCount | Long | 总商机数 |
| winBusinessCount | Long | 赢单商机数 |
| totalWinAmount | BigDecimal | 赢单总金额 |
| avgWinAmount | BigDecimal | 平均赢单金额 |
| winRateByDate | List | 按日期统计的赢单率趋势 |
| winAmountByIndustry | List | 按行业统计的赢单金额分布 |
| winAmountByCustomerLevel | List | 按客户等级统计的赢单金额分布 |

**业务逻辑**：
- 统计指定时间范围内的商机总数和赢单数量
- 计算赢单率和赢单金额统计
- 按不同维度（日期、行业、客户等级）进行分组统计

## 3. 非功能需求

### 3.1 性能要求
- 接口响应时间不超过2秒
- 支持大数据量统计（10万+商机数据）

### 3.2 数据权限
- 只能查看所属部门及子部门的数据
- 支持按用户维度查看

## 4. 数据来源

| 数据 | 来源表 | 说明 |
|------|--------|------|
| 商机数据 | crm_business | 商机基本信息、金额、结束状态 |
| 客户数据 | crm_customer | 客户行业、等级信息 |
| 用户数据 | system_user | 用户、部门信息 |

## 5. 关联接口

| 接口 | 说明 |
|------|------|
| 销售漏斗分析 | 已有的商机统计功能 |
| 业绩统计 | 已有的业绩分析功能 |

## 6. 需求优先级

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 销售预测接口 | 高 | 核心功能，需优先实现 |
| 商机赢单分析接口 | 高 | 核心功能，需优先实现 |

## 7. 验收标准

| 功能 | 验收标准 |
|------|----------|
| 销售预测 | 预测数据合理，与历史趋势相符 |
| 赢单分析 | 赢单率计算正确，各维度统计准确 |
| 数据权限 | 只能查看权限范围内的数据 |
