## 1. 产品概述

### 1.1 产品定位

MITEDTSM CRM + OA 子系统是 MIT-FMP 密讯基础平台的核心组成部分，定位为**企业级客户关系管理+办公自动化融合平台**。基于已验证的 mitedtsm 系统底座，将客户全生命周期管理与日常办公协同深度融合，为中小企业提供"管客户、管销售、管服务、管办公"的一体化解决方案。

### 1.2 产品愿景

让企业用一套系统完成客户管理、销售推进、财务跟踪、办公协同，消除CRM与OA之间的数据断层。

### 1.3 核心差异化

| 优势 | 说明 |
|------|------|
| **CRM+OA深度融合** | 销售跟进直接与审批流、任务、日程打通，不是简单的模块拼接 |
| **基于成熟底座** | 复用mitedtsm已验证的权限体系、工作流引擎、多租户架构 |
| **审批流程全覆盖** | Flowable驱动的9类审批场景，覆盖销售到财务全链路 |
| **客户360度视图** | 18个Tab标签页展示客户全维度信息 |
| **私有化部署** | 数据安全可控，支持企业内网部署 |
| **高性价比** | 中小企业可承受的价格 |

### 2.4 竞品详细分析

#### 销售易
- **优势**: CRM功能成熟，移动端体验好，大中型客户案例丰富
- **劣势**: 无HRM，价格较高，中小企业门槛高
- **差异化**: MITEDTSM融合HRM+OA，更全面的企业管理，性价比更高

#### 纷享销客
- **优势**: CRM+协同，连接型CRM理念好，BI能力突出
- **劣势**: HRM能力弱，定制化复杂，实施周期长
- **差异化**: MITEDTSM深度融合HRM，覆盖更完整的企业管理场景

#### 钉钉宜搭
- **优势**: 生态完善，协作能力强，低代码平台，用户基数大
- **劣势**: CRM深度不足，无专业销售管理能力，数据存储在公有云
- **差异化**: MITEDTSM提供专业级CRM+私有化部署

## 3. 核心价值主张

### 3.1 CRM+OA一体化

- **销售-审批-财务全链路打通**: 客户跟进→商机推进→报价→订单→审批→回款→开票，全流程在一个系统内完成
- **客户360度视图**: 18个维度Tab（基本信息、联系人、商机、订单、工单、发票、退款、费用、附件、操作日志等）
- **公海+线索池机制**: 自动掉入公海、领取限制、查重保护，确保客户资源高效流转

### 3.2 灵活审批流

- **9类审批场景**: 订单/回款/退款/报销/请假/出差/借款/请示/拜访
- **Flowable引擎**: 支持多级审批、会签、或签、条件分支、转办、加签
- **统一审批入口**: 审批人一个页面处理所有待办事项
- **超时自动催办**: 48小时未处理自动提醒+升级

### 3.3 数据驱动决策

- **销售漏斗**: 各阶段商机数量+金额+转化率，支持下钻到明细
- **销售预测**: 基于商机金额×赢单概率预测未来营收
- **业绩排名**: 按销售额/回款率/商机转化率多维度排名
- **管理驾驶舱**: 首页KPI卡片+趋势图+预警信息
- **自定义报表**: 灵活选择维度、指标、图表类型，支持定时发送

### 3.4 HR+CRM深度融合场景

| 场景 | 传统方案 | MITEDTSM融合方案 |
|------|----------|-----------------|
| 销售绩效核算 | HR系统导出薪资数据 + CRM导出业绩 → Excel手工合并 | 系统自动关联销售人员档案与CRM业绩，自动计算绩效薪酬 |
| 客户服务人员管理 | CRM分配客户 + HR排班系统独立运作 | 根据客服排班自动分配客户，服务记录关联员工档案 |
| 招聘需求分析 | 业务部门提需求 → HR独立评估 | 基于CRM业务量数据自动预测人力需求，智能发起招聘 |
| 组织架构调整 | HR系统调整 + CRM手动更新客户归属 | 组织架构调整后自动触发客户重新分配 |

### 3.5 核心差异化优势

| 优势 | 说明 |
|------|------|
| **HR+CRM深度融合** | 市场上唯一将HRM与CRM深度融合的产品，不是简单拼接 |
| **全模块覆盖** | 15个核心模块，覆盖企业90%管理需求 |
| **私有化部署** | 支持完全私有化部署，数据安全可控 |
| **高性价比** | 中小企业可承受的价格，降低数字化门槛 |
| **基于成熟底座** | 复用mitedtsm已验证的系统底座，稳定性有保障 |
| **开放生态** | 标准化API，支持与第三方系统灵活集成 |
| **渐进式采用** | 支持模块化部署，企业可按需逐步启用 |

## 5. 产品路线图

### Phase 1: CRM+OA MVP（当前）

- 客户管理（CRUD+公海+查重+联系人+导入导出）
- 商机管理（阶段推进+报价+漏斗+预测）
- 订单管理（多状态+审批流+回款+发票+退款）
- 审批中心（9类审批+统一入口+流程配置）
- 工单管理（创建+分配+处理+SLA+统计）
- 办公协作（工作报告+任务看板+日程+文档+公告）

### Phase 2: 增强与集成

- 营销工具（短信/邮件群发+模板+审核+客户关怀自动化）
- 费用+报销+借款全流程
- 移动端深度优化
- 第三方集成（企业微信/钉钉消息推送）

### Phase 3: 智能化

- AI销售预测
- 高级BI报表
- 客户画像与推荐
- 智能审批建议
