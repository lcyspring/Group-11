# CRM系统原型总体概览

## 1. 原型概述

该原型是一个企业级 **CRM + OA 综合管理系统**，位于 `/predisk/Documents/实习/Prepareation/Prototypes/PROTOTYPE/`，包含约 200+ HTML 页面，覆盖 7 大 CRM 业务域及 1 个 HR 子系统。

## 2. TOP-LEVEL NAVIGATION (from 系统框架.html)

| 序号 | 菜单模块 | 英文 | 说明 |
|------|---------|------|------|
| 1 | 客户管理 | Customer | 客户列表, 联系人, 公海客户, 客户查重 |
| 2 | 商机管理 | Opportunity | 商机列表, 销售漏斗, 预测分析 |
| 3 | 订单管理 | Order | 订单列表, 产品报价, 订单审批 |
| 4 | 财务管理 | Finance | 回款, 发票, 报销, 退款 |
| 5 | 统计报表 | Reports | 销售/财务/客户多维分析报表 |
| 6 | 工单管理 | Work Order | 工单处理 |
| 7 | 营销工具 | Marketing | 营销活动, 短信/邮件群发, 客户关怀 |
| 8 | 审批管理 | Approval | 订单/回款/报销/请假/出差/借款审批 |
| 9 | 办公管理 | Office | 公告, 任务, 日程, 请假/出差申请 |

## 3. 核心业务域划分

```
CRM 子系统:
├── 客户域 (Customer)
│   ├── 客户管理 (列表/创建/详情/公海/查重/转移)
│   ├── 联系人管理 (列表/详情/关联客户)
│   └── 客户分析 (属性/区域/成交率/星级/来源/状态)
├── 商机域 (Opportunity)
│   ├── 商机管理 (列表/创建/详情)
│   ├── 报价管理 (产品报价/确认报价)
│   ├── 销售漏斗 (阶段转化分析)
│   └── 销售预测 (预测分析/报表)
├── 订单域 (Order)
│   ├── 订单管理 (列表/创建/详情/审批)
│   ├── 产品管理 (选择产品/报价)
│   └── 合同管理 (合同编号/签署日期)
├── 财务域 (Finance)
│   ├── 回款管理 (回款/回款计划/回款审批)
│   ├── 发票管理 (开票/票据记录)
│   ├── 报销管理 (报销单/审批)
│   └── 退款管理 (退款/审批)
├── 工单域 (Work Order)
│   └── 工单管理 (列表/详情/状态流转)
├── 营销域 (Marketing)
│   ├── 营销活动 (列表/创建/详情)
│   ├── 群发工具 (短信/邮件模板+群发+审核)
│   └── 客户关怀 (生日/节假日自动祝福)

OA 子系统:
└── OA域 (Office Automation)
    ├── 审批中心 (订单/回款/报销/请假/出差/借款/客户拜访)
    ├── 办公工具 (工作报告/请示/任务/日程/文档)
    └── 申请管理 (请假/出差/借款/客户拜访)
```

## 4. 核心实体关系图

```
Customer (客户) ──1:N── Contact (联系人)
     │
     ├──1:N── Opportunity (商机) ──1:N── ProductQuotation (报价)
     │                                        │
     ├──1:N── Order (订单) ──1:N── OrderLineItem (订单产品行)
     │       ├──1:N── Invoice (发票)
     │       └──1:N── Receipt (回款)
     │
     ├──1:N── WorkOrder (工单)
     │
     └──1:N── CustomerCare (客户关怀)

Marketing:
  Campaign (营销活动) ──1:N── 目标客户
  SMSBroadcast / EmailBroadcast (群发)

OA:
  Reimbursement / Loan / Leave / Trip / Visit ── Approval (审批)
```

## 5. 通用审批状态机

```
待审批 → 审批中 → 已通过 / 被驳回 / 被否决
                           ↓
                        已撤销
```

此状态机适用于: 订单、回款、报销、退款、请假、出差、借款、客户拜访。
